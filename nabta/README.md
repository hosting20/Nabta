# نبتة 🌱 — Nabta

**منصة عربية للتحقق من الأفكار الريادية** — معالج متعدد الخطوات بدرجة نمو حية، مرشد ذكاء اصطناعي (Claude)، وتقرير تقييم شامل.

> ازرع فكرتك · تحقق منها · اجعلها تثمر

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react) ![Vite](https://img.shields.io/badge/Vite-6-646CFF?logo=vite) ![RTL](https://img.shields.io/badge/Arabic-RTL-2FA084)

## ✨ المميزات

- **٥ خطوات تحقق**: المشكلة → السوق → الحل → الإيرادات → الإطلاق
- **مؤشر نمو حي**: درجة من ١٠٠ تُحسب لحظياً بمحرك تقييم يحلل *جودة* إجاباتك (الطول، الأرقام، التحديد) وليس مجرد وجودها
- **مرشد ذكي**: رؤى مخصصة من Claude API بناءً على كامل سياق فكرتك، مع رؤى خبيرة احتياطية تعمل بدون إنترنت
- **تقرير نهائي**: نقاط قوة، مخاطر، خطوات تالية، وحكم نهائي
- **حفظ تلقائي**: إجاباتك محفوظة في المتصفح (localStorage)
- **تصميم فاتح أنيق** بباليتة خضراء: `#EEEEEE · #6FCF97 · #2FA084 · #1F6F5F`

## 🚀 التشغيل

```bash
npm install
npm run dev        # http://localhost:5173
```

### تفعيل الذكاء الاصطناعي (اختياري)

بدون مفتاح، يعمل التطبيق كاملاً برؤى الخبراء المحفوظة. لتفعيل Claude:

```bash
cp .env.example .env
# ضع مفتاحك من https://console.anthropic.com
```

> ⚠️ الاستدعاء المباشر من المتصفح مناسب للتجربة فقط. في الإنتاج استخدم خادماً وسيطاً (proxy) حتى لا يظهر مفتاحك للزوار — مثلاً Cloudflare Worker أو Vercel Edge Function تمرر الطلب لـ `api.anthropic.com`.

## 🔢 كيف يُحسب التقييم؟

كل نص يُقيَّم من 0 إلى 1 على أربعة محاور: **الطول الكافي** (0.4) + **وجود أرقام** (0.2) + **كلمات التحديد** مثل "شهرياً، مثال، $" (0.25) + **أساس الوجود** (0.15) − **عقوبة العمومية** مثل "الجميع". ثم تُجمع الأبعاد بأوزان Lean Startup:

```
الدرجة = المشكلة×30% + السوق×25% + الحل×25% + الإيرادات×20%
```

| الدرجة | المرحلة |
|--------|---------|
| 0–39 | 🌱 بذرة |
| 40–69 | 🌿 نبتة تنمو |
| 70–100 | 🌳 شجرة مثمرة |
